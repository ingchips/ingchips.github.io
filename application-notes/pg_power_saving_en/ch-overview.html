Programmer’s Guide - Power Saving

3 Overview

The implementation of power saving in SDK is designed to be easy to use. Generally, power saving relies on the idle process in RTOS, which is usually a process3 having the lowest priority. It’s a common practice, for example, there is a System Idle Process in Windows with PID 0.
