招贤纳士

投送简历 时请注明所感兴趣的职位。

嵌入式软件工程师（蓝牙应用）

岗位职责

  1. 开发BLE产品的嵌入式系统软件，包括通信控制和驱动程序等；
  2. 对BLE系统硬件、电路设计提出指导建议；
  3. 协同完成BLE系统设计、调试和测试。

任职资格

  1. 电子、计算机、通信相关专业本科及以上，精通C/C++语言编程；
  2. 悉ARM Cortex-M系列处理器及开发环境，了解UART、I2C、SPI等接口；
  3. 熟悉BLE相关Profile、Mesh开发者优先；
  4. 有iOS/Android移动应用开发经验（Xamarin/Web）者优先；
  5. 良好的英语写作能力，口语佳者优先；
  6. 有团队精神，良好的沟通能力及工作主动性。

通信协议工程师（低功耗蓝牙）

岗位职责

  1. 负责/参与BLE协议栈（Controller/Host/Mesh）的开发；
  2. 负责/参与BLE协议栈（Controller/Host/Mesh）自动化测试平台的开发；
  3. 与ASIC工程师协同完成系统集成和联合调试。

任职资格

  1. 电子、计算机、通信相关专业本科以上，精通C/C++语言编程；
  2. 精通BLE、Wifi、ZigBee、LTE等无线通信协议中的一种或多种，有一年以上协议栈开发经验；
  3. 熟悉Python或Erlang语言优先；
  4. 良好的英语写作能力，口语佳者优先；
  5. 有团队精神，良好的沟通能力及工作主动性。
