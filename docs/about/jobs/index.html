招贤纳士

投送简历 时请在邮件标题注明所感兴趣的职位。 以下职位工作地点可选择北京、深圳。

通信协议工程师

招聘人数：2~3 名

岗位职责
  1. 负责/参与 Bluetooth LE 协议栈（Controller/Host/Mesh）的开发；
  2. 负责/参与 Bluetooth LE 协议栈（Controller/Host/Mesh）自动化测试平台的开发；
  3. 与 ASIC 工程师协同完成系统集成和联合调试。
任职资格
  1. 电子、计算机、通信相关专业本科以上，精通C/C++语言编程；
  2. 精通 Bluetooth LE、WIFI、ZigBee、LTE 等无线通信协议中的一种或多种，有一年以上协议栈开发经验；
  3. 熟悉 Python 或 Erlang 语言优先；
  4. 有团队精神，良好的沟通能力及工作主动性。

音频工程师

招聘人数：1 名

岗位职责

  1. 负责/参与 LC3 音频编解码器的开发和优化；
  2. 负责/参与 Bluetooth LE Audio 协议栈的集成；
  3. 与 ASIC 工程师协同完成音频系统集成和联合调试。

任职资格

  1. 电子、计算机、通信相关专业本科及以上，精通 C/C++ 语言编程；
  2. 熟悉 SBC、Opus、EVS 等音频解码器的一种或多种；
  3. 具备定点算法开发及优化经验；
  4. 有团队精神，良好的沟通能力及工作主动性。

嵌入式软件工程师（蓝牙应用）

招聘人数：4~5 名

岗位职责

  1. 为 Bluetooth LE 产品开发嵌入式系统软件，包括通信控制和驱动程序等；
  2. 对 Bluetooth LE 系统硬件、电路设计提出指导建议；
  3. 协同完成 Bluetooth LE 系统设计、调试和测试；
  4. 分析定位客户问题，提出解决方案。

任职资格

  1. 电子、计算机、通信相关专业本科及以上，精通C/C++语言编程；
  2. 熟悉 ARM Cortex-M 系列处理器及开发环境，了解 UART、I2C、SPI 等接口；
  3. 熟悉 Bluetooth LE 相关 Profile、Mesh 开发者优先；
  4. 有团队精神，良好的沟通能力及工作主动性。

移动端开发工程师

招聘人数：1~2 名

职位职责

  1. 负责移动端跨平台应用开发；
  2. 配合蓝牙芯片功能，设计 App 架构并完成功能开发；
  3. 分析定位客户问题，提出解决方案。

任职资格

  1. 1 年及以上移动端软件开发经验，有大型互联网公司独立负责移动端产品开发经验优先；
  2. 必须有 Android、iOS 或者小程序开发经验；
  3. 熟悉以下语言中的 2-3 项：Java/Swift/Javascript/Kotlin/C#；
  4. 有 Bluetooth LE 相关应用开发经验者优先；
  5. 其他领域加分项：前端开发、数据库开发；
  6. 有较强的自驱力。

IC设计工程师（通信协议、算法实现）

岗位职责

  1. 负责通信协议、算法实现（BLE/WIFI/UWB等MAC/PHY）
  2. 设计文档撰写、代码质量检查
  3. 与软件和算法工程师协同完成模块定义、软硬件划分、系统调试
  4. 配合验证工程师完成模块验证和代码覆盖率检查

任职资格

  1. 微电子、电子、计算机、通信相关专业本科以上
  2. 熟练使用Verilog/SystemVerilog进行逻辑设计
  3. 精通 BT、BLE、WIFI、ZigBee、UWB、蜂窝等无线通信协议中的一种或多种
  4. 有团队精神，良好的沟通能力及工作主动性

IC设计工程师（SoC设计、集成、实现）

岗位职责

  1. 设计集成数字接口模块（I2C/I2S/UART/SPI/USB等）
  2. 设计时钟复位系统和控制系统
  3. 模块级和系统级低功耗设计
  4. 代码质量检查
  5. 制定时序约束和低功耗设计约束
  6. 与软件工程师协同完成系统集成和联合调试

任职资格

  1. 微电子、电子、计算机、通信相关专业本科以上
  2. 熟练使用Verilog/SystemVerilog进行逻辑设计
  3. 熟悉数字前端流程，清晰理解概念
  4. 具有逻辑综合、STA、形式验证、低功耗检查、DFT等能力优先
  5. 有团队精神，良好的沟通能力及工作主动性

IC设计工程师、经理（模拟、射频）

岗位职责

  1. 设计CMOS模拟和射频IC电路，包括LDO,DCDC,ADC/DAC, PLL, LNA, MIXER,PA,IO等模块
  2. 负责电路模块级结构选择和电路设计
  3. 负责模块验证和芯片测试工作
  4. 负责指导并参与版图设计

任职资格

  1. 微电子、电子相关专业本科以上
  2. 熟悉常用模拟模块及数模混合电路设计
  3. 熟悉CMOS射频电路设计
  4. 具有带团队经验优先
  5. 有团队精神，良好的沟通能力及工作主动性

IC验证工程师

岗位职责

  1. 维护开发验证平台
  2. 模块级和芯片级的验证
  3. 编写验证计划以及报告
  4. 提高验证质量和效率
  5. 芯片回片测试

任职资格

  1. 电子、计算机、通信相关专业本科以上
  2. 精通UVM/VMM
  3. 精通C语言和脚本语言（Perl、Python、Makefile等）
  4. 熟悉VCS、iRUN等相关EDA工具
  5. 有完整和成功的芯片验证经验的优先
  6. 有团队精神，良好的沟通能力及工作主动性

硬件工程师

岗位职责

  1. 硬件板级的开发及调试
  2. 硬件BT/BLE/WIFI常规的射频及音频测试
  3. 客户硬件支持
  4. 硬件相关文档撰写及整理

任职资格

  1. 电子、计算机、通信相关专业本科以上
  2. 熟悉PCB及原理图设计
  3. 熟悉常用仪器使用，频谱仪、信号源、BT/BLE/WIFI相关的测试仪器等
  4. 熟悉常规的BT/BLE/WIFI射频测试及音频方面的测试
  5. 动手能力强，可以做常规的PCB元器件焊接等
  6. 有手机和BT/BLE/WIFI工作经验的优先
  7. 有团队精神，良好的沟通能力及工作主动性

产品质量工程师

岗位职责

  1. 组织协调公司产品测试，客户产品测试，客户产线测试
  2. 定期客户拜访，追踪客户质量
  3. 客诉处理，组织协调公司内部的分析调查，负责向客户报告
  4. 负责客户稽核要求的准备和执行，以及后续问题的跟踪结案
  5. 配合公司研发，负责客户端各种试产的信息收集
  6. 积极维护客户关系，及时准确反馈客户端信息，做好客户和公司的信息沟通桥梁
  7. 宣导客户遵循公司相关产品的设计使用提醒或建议事项
  8. 定期整理产品端，客户端问题，从客户端角度push公司内部解决问题

任职资格

  1. 电子、管理等相关专业本科以上
  2. 具有半导体行业工作经验，熟悉半导体供应商生产制程及质量管控，有半导体design house、封测厂的工程质量工作经验优先
  3. 具有一定的FA分析能力
  4. 具有较强的逻辑去发现问题、并分析和处理的能力，具备较强的沟通和协调能力
  5. 良好的英语听、说、读、写能力，能与客户英文沟通
  6. 有良好的执行能力和团队合作意识
