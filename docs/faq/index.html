FAQ

如何快速评估射频性能？

  • 方法 1：使用蓝牙综测仪评估

    1. 通过 ingWizard 主菜单 Test → Download BQB RF Test Binary 下载 BQB 测试程序；

    2. 将串口连接到蓝牙综测仪进行测试。

  • 方法 2：使用频谱仪评估

    1. 通过 ingWizard 主菜单 Test → Download BQB RF Test Binary 下载 BQB 测试程序；

    2. 通过 ingWizard 主菜单 Tools → More → BQB RF Test 打开测试工具；

    3. 通过 BQB RF Test 工具控制设备发送 CW 信号，使用频谱仪观察信号强度、频偏。

  • 方法 3：使用开发板做简易评估
